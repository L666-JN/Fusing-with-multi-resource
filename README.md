# 多源数据匹配融合研究项目

## 项目概述
本项目专注于海上多源传感数据的匹配与融合研究，集成雷达、AIS（船舶自动识别系统）和北斗（BDS）等多源数据，实现船舶目标的综合跟踪与识别。

## 技术特点

1. **多源数据集成**：集成雷达、AIS、北斗三种传感数据源
2. **数据融合算法**：采用加权融合算法实现多源数据综合处理
3. **实时可视化**：提供实时数据查看和轨迹回放功能
4. **地理坐标转换**：支持不同坐标系之间的转换
5. **数据回放机制**：支持历史数据的回放与分析

## 主要功能模块

### 核心融合模块 (src/core/weighted_fusion.py)
- 实现多源数据的加权融合算法
- 处理不同传感器数据的时空对齐
- 生成融合后的综合轨迹数据

### 数据摄入模块 (src/ingestion/replayer.py)
- 支持多源数据的读取和解析
- 实现数据的时间同步处理
- 提供数据回放功能用于算法验证

### 可视化模块
- **realtime_viewer.py**: 实时数据可视化显示
- **visualize_tracks.py**: 轨迹数据的可视化分析

### 工具模块
- **geo_transform.py**: 地理坐标系统转换
- **radar_json_to_csv.py**: 雷达数据格式转换
- **remove_bds_mmsi.py**: 北斗数据清洗工具

## 数据流程

1. **原始数据采集**: 从雷达、AIS、北斗等传感器获取原始数据
2. **数据预处理**: 格式转换、数据清洗、坐标统一
3. **数据匹配**: 基于时空特征进行多源数据匹配
4. **数据融合**: 采用加权算法进行数据融合
5. **结果输出**: 生成融合后的轨迹数据
6. **可视化分析**: 提供实时查看和轨迹回放功能
